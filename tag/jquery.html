Visualizing pnr changes over time using flot.js

Tue 03 December 2013
By Salil Panikkaveettil

In tech.

tags: flotpnrgraphJQuery

Around 2 weeks back, pnr prediction tool got heavy criticism from a user. He argued that there was no prediction algorithm being used and the result was just a random prediction. We tried to explain how our algorithm worked and how we use curve fitting to arrive at a reasonably accurate prediction. We also showed him the accuracy of our pnr prediction and argued that we cannot achieve such a high accuracy by writing any random function. But it was tough to convince him and to his credit, we were not showing any data that we used for prediction in the website.

A decision was made to show the user the data we use for prediction. This will help users take an informed decision rather than blindly trusting pnr.me. If the user is adventurous enough he can arrive at his own prediction using the data

Data

The data we want to share with the user is the exact data that we use for prediction. From our database, we select 15 pnr numbers which match the characteristics of the pnr being queried. Each of these pnrs will have information about how the waiting list changed over time and our prediction tool uses this data to arrive at a prediction.

Visualization

The pnr change data for a pnr is a set of points where x co-ordinate represents the number of hours before departure and y co-ordinate represents the waiting list number. We decided to plot all the 15 curves in a graph

Technologies used

Our initial idea was to use D3.js as it is a very powerful tool to visualize data. But we were already using JQuery and it made much more sense to use a plugin for JQuery than another complete JS library. Enter flot.js.

Using flot was very simple and here is the sample code we used
